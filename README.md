## 📖 Sobre el portafolio

Este portafolio trato de mantenerlo siempre actualizado con la información más reciente sobre mi, espero pueda servirles de inspiración o más bien para conocerme más. No quiero explayarme mucho acá ya que quizás es repetir información que pueden encontrar dentro del mismo.

## 💻 Tecnologías usadas

- **Framework:** React.js
- **UI:** Chakra-UI
- **Gestor de paquetes:** Yarn

## 🚀 ¿Cómo deployarlo?

```bash
git clone https://github.com/DarioAlbor/Portfolio.git
cd Portfolio
yarn
yarn start
```
