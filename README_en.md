- **Framework:** React.js
- **UI:** Chakra-UI
- **Package Manager:** Yarn

## 🚀 How to Deploy

```bash
git clone https://github.com/DarioAlbor/Portfolio.git
cd Portfolio
yarn
yarn start
```
